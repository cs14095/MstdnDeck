# MstdnDeck

![MstdnDeck_icon](https://github.com/shoponpon/MstdnDeck/blob/master/img/icon.png?raw=true)

This is a Simple Dedicated Web browser of Mastodon.

Can open Mastodon like the TweetDeck App for OSX.
